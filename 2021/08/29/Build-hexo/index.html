Now you may want to take this project online.
The easiest way for starters would be moving the application to Github repository.

Assuming you already have a github account, then we can got to next step.

create an repository, and name it
${your username}.github.io

next,

then generate your github token as your password.

Now you can visit your blog by connecting to $<your username>.gitbub.io

Note: If it doesn’t display correctly, go to

Make sure your site is published and source is changed to master.

And also change theme because it might set to default.
